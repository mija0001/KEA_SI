Access instructions

Access from command line

The database may be accessed from a command line terminal with:

>>> mysql -u [USERNAME] -h [DATABASE_IP] -p

enter the password when prompted

You may need to install the MySQL client first:

Unix/Linux: Open terminal and enter: sudo apt install mysql-client

Access from GUI

The database may also be accessed via a GUI like MySQL Workbench

Access from library/module

The database may also be accessed using a library like MySQL-python

Access level for user

Table: invoices

User has SELECT, INSERT, UPDATE and DELETE privileges on the table.

Table: customers_integrator

User has SELECT privileges on the table.

This is a limited view of the customers table, which does not include the 'phone' field.

Other databases, tables & privileges

User does not have access to the 'employees' table, or the full 'customers' table.

User does not have access to other databases.

User does not have priviliges to grant privileges.

User does not have privileges to create, alter or delete users, databases, tables or views
